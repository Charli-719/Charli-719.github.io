3、flex

利用弹性布局使元素居中。给父元素设置

justify-content:center;//水平居中
align-items:center;//垂直居中

4、table-cell

组合使用display:table-cell和vertical-align、text-align，使父元素内的所有行内元素水平垂直居中。适用于不知道元素具体宽和高是。
